## Upgrade:

### Homebrew

```ruby
brew upgrade warrensbox/tap/tgswitch
```
### Linux

Rerun:

```sh
curl -L https://raw.githubusercontent.com/warrensbox/tgswitch/release/install.sh | bash
```

## Uninstall:

### Homebrew

```ruby
brew uninstall warrensbox/tap/tgswitch
```
### Linux

Run:

```sh
rm /usr/local/bin/tgswitch
```
